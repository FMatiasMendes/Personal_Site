CONSTRUCTION COMPANY WEBSITE

FINAL COURSE PROJECT

As part of the Full Stack Web Development course that I've did at Codemaster, I had the opportunity to create a comprehensive website for a fictional construction company. This project served as my final assessment and involved adhering to guidelines provided by the web design department to ensure a professional and user-friendly outcome.

The website features a clean and modern design, showcasing various projects and services offered by the construction company. A key component of this project was the development of a back office site, allowing the client to update site information, add news articles, and manage new projects effortlessly.

For the frontend I used HTML, CSS, Bootstrap and JavaScript to ensure a responsive user experience. The backend was built with PHP and MySQL to handle dynamic content efficiently.

This project enhanced my technical skills and reinforced the importance of user experience and effective client communication in web development. I am proud of the outcome and eager to apply my knowledge to future projects!
